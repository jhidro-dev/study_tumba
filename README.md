## Pages & Features

- **index.html**: Introduction, semantic layout, navigation, anchor links, image + text.
- **about.html**: About article, blockquote, address, citations.
- **services.html**: Ordered/unordered/definition lists, full HTML table.
- **contact.html**: Contact info, `mailto:` link, embedded Google Map (iframe), embedded content.
- **form.html**: Full HTML form with many input types, select/optgroup, datalist, output.
- **media.html**: Image gallery, audio, video, figure + caption, image map.
